[center][size=12pt][u][b]Compatibility[/b][/u][/size][/center]

The "[i]Justify BBCode[/i]" is compatible for only SMF [b]2.0.X[/b]. It will soon not be updated to 1.1.X as 2.0 is the current development version.

[center][size=12pt][u][b]Features[/b][/u][/size][/center]

The Justify BBCode modification adds a BBCode after the existing alignment BBCodes on the post page. You can use the [justify] BBCode to use this tag. The Justify BBCode uses a DIV Text Alignment to justify the text using pure HTML.

[center][size=12pt][u][b]Installation[/b][/u][/size][/center]

Install this modification through the package manager. It only extracts one image file, and edits two (besides the language files), so nothing should go wrong.
